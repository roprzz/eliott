# Eliott ♥

01.12.2025

I made this for my son.

Eliott was born on December 1st, 2025. I wanted to leave him something that
could last, so I put his name in a little program that lights up dot by dot,
like the old machines I grew up with. The plan is to burn it on a CD and keep
it somewhere safe until he's old enough to wonder what it is.

Maybe CDs will be impossible to read by then. Honestly, that's part of why I
like the idea. He'll have to dig a little, find an old drive somewhere, and
when he finally opens it, all he'll find is his name, his birthday and a
heart. That's it. That's the whole thing.

Eliott, if you're reading this one day: it works, I checked. I love you.

## What it does

Open `dist/index.html` in any browser. Everything is packed in that one file,
no internet needed. A little terminal boots up, pretends to read the disc, and
writes:

**Eliott. 01.12.2025. ♥**

## If you want to tinker

```bash
npm install
npm run dev
npm run build
```
